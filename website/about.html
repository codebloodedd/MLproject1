About

Pratham is a graduate student focused on data science and software engineering, with interests in machine learning, analytics, and building reliable data pipelines. This project uses real-world movie metadata to explore what “success” looks like across popularity and audience ratings.

The project emphasizes transparency and repeatability by linking to the raw data, cleaned data, and all Python code used for collection, cleaning, and visualization. Future modules will add PCA, clustering, association rule mining, and supervised modeling to deepen the analysis.
